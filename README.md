# GentGranBD

Aplicació d'escriptori per a la gestió de l'Associació Gent Gran de Castelldefels.

## Llicència

Aquest projecte es distribueix sota la llicència MIT. Consulta `LICENSE`.

## Actualitzacions

Les versions es publiquen com a GitHub Releases del repositori principal. La CI genera els binaris de Windows i macOS, els fitxers `.sha256` corresponents i els adjunta al release.

L'aplicació comprova actualitzacions contra el release més recent. Si el repositori és públic no cal token; si es torna privat, es pot configurar `GENTGRAN_GITHUB_TOKEN`.
